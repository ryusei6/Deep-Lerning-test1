# Predict_Nikkei_Stock_Average
Predict the next closing price of the Nikkei Stock Average.

python 3.7.1

## Install
```
(venv)$ pip install -r requirements.txt
```


## Usage  
### Train model
```
(venv)$ python train_model.py
```

### Predict the next closing price of the Nikkei Stock Average
```
(venv)$ python predict_price.py
```
